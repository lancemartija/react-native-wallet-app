## Run Locally

Clone the project

```bash
  git clone https://github.com/lancemartija/react-native-wallet-app
```

Go to the project directory

```bash
  cd react-native-wallet-app
```

Install dependencies

```bash
  npm install
```

Start the expo server.

```bash
  npx expo start
```
or
```bash
  npx expo start --tunnel
```
